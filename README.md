# Private Academy - Industrial GTM

This private repository contains implementation-oriented academy modules for industrial B2B GTM operators.

It is designed for:

- internal agency employees;
- clients and their teams;
- AI agents built and deployed by the agency.

The academy exists to train people and agents to build industrial go-to-market systems that create trust, account movement, buyer enablement, and qualified pipeline in long-cycle manufacturing and export markets.

## Operating Philosophy

This is not a passive course library.

Each module must produce field-ready artifacts that can be inspected by:

- sales;
- marketing;
- RevOps;
- leadership;
- technical or SME teams;
- AI agents.

## Non-Negotiables

- No generic SaaS marketing advice.
- No vanity metrics as primary proof.
- No named-company case studies.
- No AI output accepted without human correction.
- No sales activation before demand state, account fit, role, and signal strength are interpreted.
- No pilot scale decision without weekly cadence evidence.

## Source Boundary
