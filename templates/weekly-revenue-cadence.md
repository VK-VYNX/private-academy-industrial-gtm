# Weekly Revenue Cadence

Use this template for the weekly sales-marketing GTM meeting.

## Meeting Purpose

Move target accounts through the GTM system by reviewing account progression, buyer signals, sales feedback, content needs, and next actions.

## Attendees

- Sales owner
- Marketing owner
- Leadership sponsor
- Technical or product expert when needed
- Customer success or service representative when installed-base learning matters

## Agenda

### 1. Progression Wins

- Which accounts moved forward?
- What caused the movement?
- What should we repeat?

### 2. Active Focus Review

| Account | Current Play | Buying Committee Coverage | Risk | Next Action |
|---|---|---|---|---|
|  |  |  |  |  |

### 3. Future Pipeline Review

| Account | Signal | Relationship Path | Nurture Asset | Next Action |
|---|---|---|---|---|
|  |  |  |  |  |

### 4. New Signals

- first-party signals;
- third-party signals;
- ecosystem signals;
- sales-discovered signals.

### 5. Content And Proof Needs

| Need | Buyer Role | Demand State | Owner | Due Date |
|---|---|---|---|---|
|  |  |  |  |  |

### 6. Stalled Accounts

- Which accounts are stuck?
- Is the issue fit, timing, access, proof, budget, competition, or internal consensus?
- Should we continue, change play, or disqualify?

### 7. Metrics

- account movements this week;
- new future pipeline accounts;
- new active-focus accounts;
- opportunities created;
- buying committee coverage;
- content engagement from target accounts;
- pipeline velocity movement.
