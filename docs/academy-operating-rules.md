# Academy Operating Rules

## Purpose

The Private Academy exists to operationalize industrial GTM intelligence into training, implementation, and AI-agent workflows.

Every module must help learners produce a real operating artifact, not merely understand a concept.

## Module Standards

Each module should include:

- learner-facing guide;
- facilitator runbook;
- workbook or worksheet pack;
- AI-agent prompt library;
- visual flowcharts;
- assessment rubric;
- capstone artifact;
- quality gates;
- implementation checklist or sprint plan;
- anonymized composite examples.

## Learning Design Rhythm

Every unit should include:

1. concept;
2. industrial example;
3. diagnostic question;
4. individual exercise;
5. group critique;
6. AI-assisted attempt;
7. human correction;
8. final artifact.

## AI Governance

AI agents may:

- draft;
- classify;
- compare;
- critique;
- assemble artifacts;
- flag risks.

AI agents must not:

- invent named-company examples;
- fabricate source evidence;
- treat engagement as buying intent;
- optimize for vanity metrics;
- bypass human correction;
- recommend scale before pilot evidence exists.

## Confidentiality

Use anonymized composite examples only.

Do not name restricted real companies, private clients, or confidential inspirations.
