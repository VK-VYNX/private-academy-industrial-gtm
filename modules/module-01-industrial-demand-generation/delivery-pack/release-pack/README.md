# Release Pack

This pack controls Module 1 release decisions.

## Files

| File | Purpose |
|---|---|
| [module-release-checklist.md](module-release-checklist.md) | release gate checklist |
| [module-version-log.md](module-version-log.md) | version history and release notes |

## Release Standard

Module 1 is release-grade only when:

- automated validation passes;
- visual render QA is reviewed;
- calibration notes are current;
- cohort feedback has been inspected;
- restricted-name scan has been run with the current local list;
- the release checklist has an owner and decision.

## Release Owner

The academy owner approves release status. Facilitators, reviewers, RevOps operators, and AI operators provide evidence.
