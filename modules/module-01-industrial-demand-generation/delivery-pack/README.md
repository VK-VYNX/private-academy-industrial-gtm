## What This Pack Is For

This pack converts Module 1 into four usable experiences:

- self-paced learner path;
- live workshop path;
- client implementation sprint;
- AI-assisted operating workflow.

## What This Pack Is Not

- It is not a generic marketing course.
- It is not a lead-generation checklist.
- It is not a passive reading packet.
- It is not a named-company case-study library.

Every section should push the learner toward an operating artifact that sales, marketing, RevOps, leadership, and AI agents can inspect.
