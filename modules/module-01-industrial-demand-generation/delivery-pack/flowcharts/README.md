## Visual Design Rules

- Every diagram should change behavior, not decorate the course.
- Every diagram should make sales, marketing, RevOps, leadership, and AI-agent implications visible.
- Decisions should be diamonds.
- Operating artifacts should be rectangles.
- Failure loops should route back to repair, not forward to scale.
- No named-company examples.
