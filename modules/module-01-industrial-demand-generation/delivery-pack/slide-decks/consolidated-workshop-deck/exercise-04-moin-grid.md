## Choose From These Options

Use one demand state for each question.

| Demand State | Buyer Is Asking |
|---|---|
| Content Demand | What problem, risk, opportunity, or cost of inaction matters? |
| Solution Demand | Which approach, requirement, implementation path, or tradeoff fits? |
| Vendor Demand | Which supplier can prove value, reduce risk, and support the purchase? |

Use these starter question patterns.

| Role Type | Buyer Question Pattern |
|---|---|
| Technical | Will this work in our application or specification? |
| Production or Operations | Will this improve output without creating disruption? |
| Quality or Compliance | Can we validate, document, inspect, or approve this safely? |
| Maintenance or Service | Will this be serviceable and supported over time? |
| Procurement | Can this supplier meet our comparison and onboarding requirements? |
| Finance or Leadership | Is the cost, risk, payback, or priority justified? |
| Champion | Can I explain this internally when the supplier is not present? |
