# Exercise 02 - ICP And Focus Statement Worksheet

Use this worksheet with Slide 11 of the Industrial Demand Generation workshop.

This exercise helps you choose one practical 90-day ICP focus for an industrial product or product category. You are not expected to write an ICP from scratch. For each parameter, choose the statement that best matches the candidate segment.

## Exercise Outcome

By the end of this exercise, you will have:

- One selected 90-day ICP focus;
- A clear reason for choosing that focus;
- Multi-select disqualification logic for accounts that should be routed, researched, nurtured, downgraded, or rejected.

The output becomes the ICP and disqualification section of your Industrial Demand Generation strategy.

## How To Complete This Exercise

1. Use the Candidate Segment Builder to assemble two or three candidate segments.
2. Score each candidate segment using the parameter options.
3. Choose the segment with the strongest combination of fit, trigger, access, value, proof, and pilot feasibility.
4. Write one 90-day ICP focus statement.
5. Select all disqualification rules that apply. This section is multi-select because more than one rule may be true at the same time.

Do not choose the largest market automatically. Choose the segment where the next 90 days can create useful account progression and learning.

Use this phrasing formula:

```text

```

Avoid broad labels such as "all manufacturers," "all OEMs," "all exporters," or "any plant with budget."

## Scoring Ladder

| Score | Status | Plain Meaning |
|---:|---|---|
| 1 | Weak Or Unclear | The segment is too broad, speculative, or difficult to act on in a 90-day pilot. |
| 2 | Possible But Loose | The segment has some logic, but fit, timing, access, or proof is still too vague. |
| 3 | Promising But Incomplete | The segment is plausible, but at least one important operating condition is missing. |
| 4 | Strong And Usable | The segment is specific enough to guide content, distribution, sales action, and signal routing. |
| 5 | Pilot-Ready | The segment has strong fit, visible trigger, reachable accounts, useful proof, and clear operating ownership. |

## Multi-Select Disqualification Logic

This section is multi-select. More than one rule may apply to the same account.

Use the strongest applicable routing decision:

1. If a Reject rule applies, reject the account from the 90-day pilot.
2. If no Reject rule applies but a Downgrade rule applies, downgrade the account.
3. If the account may become relevant later, nurture it.
4. If evidence is missing but the account may be valuable, research it before routing.

## Reject When

Select all that apply.

| Select | Reject Rule |
|---|---|
|  | Wrong process, application, material, capacity, compliance level, or operating environment. |
|  | Expected value, margin, repeat potential, or service potential is too low for focused effort. |
|  | Buyer only wants a lowest-price comparison with no interest in proof, fit, risk, or value. |
|  | No relevant trigger exists and there is no evidence of timing, urgency, or active problem. |
|  | The account is outside the selected geography, service area, channel model, or delivery capability. |
|  | The required proof, certification, reference, technical validation, or service capability cannot be credibly provided. |
|  | The account is a one-off opportunity that would distract from the 90-day ICP focus. |

## Downgrade When

Select all that apply.

| Select | Downgrade Rule |
|---|---|
|  | Fit is possible, but trigger evidence is weak or timing is unclear. |
|  | The account matches the broad segment but lacks enough technical or commercial fit for priority attention. |
|  | Procurement or a junior contact is visible, but the real buying committee is not yet accessible. |
|  | The account may be valuable, but proof gaps make immediate sales activation risky. |
|  | Engagement exists, but the signal is weak, isolated, old, or not tied to a clear role. |
|  | Sales effort would be high relative to the current evidence of movement. |

## Nurture When

Select all that apply.

| Select | Nurture Rule |
|---|---|
|  | The account fits the segment but is in early learning mode. |
|  | The trigger may emerge later, such as future expansion, replacement cycle, audit, tender, or budget window. |
|  | The account needs education before solution or vendor evaluation is realistic. |
|  | The committee is not ready, but the segment is strategically relevant. |
|  | The account is not part of the 90-day priority set but should remain in Future Pipeline. |

## Research When

| Select | Research Rule |

|  | Trigger evidence is suspected but not yet confirmed. |
