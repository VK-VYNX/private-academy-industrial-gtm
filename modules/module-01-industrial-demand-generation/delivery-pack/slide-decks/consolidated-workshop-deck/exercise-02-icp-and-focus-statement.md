# Exercise 02 - Ideal Customer Profile and Focus Statement Worksheet

Use this worksheet with Slide 11 of the Industrial Demand Generation workshop.

Ideal Customer Profile (ICP) means the type of account most worth focused effort. This exercise helps you choose one practical 90-day ICP focus for an industrial product or product category.

You are not expected to write an expert strategy from scratch. You will compare two or three possible segments and choose the one most likely to create useful learning and account movement in 90 days.

## Exercise Outcome

By the end of this exercise, you will have:

- One selected 90-day ICP focus;
- A clear reason for choosing that focus;
- Multi-select rules for accounts to prioritize, research, nurture, downgrade, or reject.

The output becomes the Ideal Customer Profile and Disqualification section of your Industrial Demand Generation strategy.

## How to Complete This Exercise

1. Build two or three candidate segments using the option menus.
2. Score each segment with five practical checks.
3. Choose the strongest 90-day focus.
4. Write the focus statement.
5. Select all rules that apply for prioritize, research, nurture, downgrade, or reject.

Do not choose the largest market automatically. Choose the segment where the next 90 days can create the most useful learning and movement.

## 90-Day ICP Focus Statement

Assemble the focus statement from your selected options.

```text
For the next 90 days, we will focus on [Account Type] in [Sector or Application] within [Geography or Market] where [Trigger] creates a likely need for [Operational Outcome].

```

## Multi-Select Action Rules

This section is multi-select. More than one rule may apply to the same account.

Use the strongest applicable decision:

1. Reject if the account clearly does not fit.
2. Downgrade if the account may fit later but should not get priority now.
3. Nurture if the account fits but is too early.
4. Research if important evidence is missing.
5. Prioritize if fit, trigger, access, and proof are strong.

|  | Proof availability is unclear. |

## Nurture When

Select all that apply.

| Select | Nurture Rule |
|---|---|
|  | Account fits the segment but is still learning. |
|  | Trigger may appear later, such as future expansion, replacement, audit, tender, or budget window. |
|  | Buyer needs education before solution or supplier evaluation. |
|  | Account is strategically relevant but not urgent in the next 90 days. |
