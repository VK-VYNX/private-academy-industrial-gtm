## Choose From These Options

Choose five success measures only.

| Success Measure | Use When |
|---|---|
| Target Accounts Reached | The pilot must reach the right companies. |
| High-Fit Accounts Engaged | Fit matters more than raw traffic. |
| Buying Roles Identified | The committee is becoming more visible. |
| Buyer Questions Answered | Content and proof are reducing uncertainty. |
| First-Five Assets Completed | The asset system is becoming usable. |
| Trusted Paths Used | Distribution is happening through real paths. |
| Useful Sales Conversations Created | Sales is getting better reasons to engage. |
| Low-Fit Accounts Avoided | The team is protecting effort. |

Use these decision meanings.

| Decision | Use When |
|---|---|
| Stop | The focus, proof, access, or sales usefulness is too weak. |
| Repair | There is useful evidence, but one part of the system needs fixing. |
| Scale | Target accounts are moving and the system is working consistently. |
