# Exercise 01: Demand System Diagnostic

## Purpose

Diagnose whether the current motion is creating trust and account movement, or only collecting activity and leads.

## Inputs

- Current revenue or pipeline problem.
- Current channels and content.
- Current sales feedback.
- Current CRM or dashboard view, if available.

## Instructions

1. Choose one industrial category.
2. Fill the system diagnostic table.
3. Identify the strongest current layer.
4. Identify the weakest current layer.
5. Define one repair action that can be tested within 30 days.
