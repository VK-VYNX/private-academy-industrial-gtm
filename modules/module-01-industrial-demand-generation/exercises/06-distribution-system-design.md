# Exercise 06: Distribution System Design

## Purpose

Create a distribution system that makes content visible through trusted industrial channels.

## Inputs

- Content plan.
- Channel access.
- Sales team capacity.
- Partners, distributors, associations, or events.

## Instructions

1. Select distribution channels by buyer trust and demand state.
2. Assign owners.
3. Define cadence.
4. Define account-level signals.
5. Define follow-up rules.
