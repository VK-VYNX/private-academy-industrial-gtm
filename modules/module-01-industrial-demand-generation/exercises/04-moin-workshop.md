# Exercise 04: MOIN Workshop

## Purpose

Turn buyer uncertainty into a role-specific MOIN grid.

## Inputs

- ICP definition.
- Buying committee map.
- Sales objections.
- Existing content list.

## Instructions

1. Write buyer questions by role.
2. Classify each question by demand state.
3. Identify missing assets.
4. Mark which assets sales can use.
5. Select the first five assets to create or repair.
