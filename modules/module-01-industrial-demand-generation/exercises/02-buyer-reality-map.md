# Exercise 02: Buyer Reality Map

## Purpose

Map the buying committee and identify role-specific proof needs.

## Inputs

- Selected category.
- Typical buyer roles.
- Known objections.
- Sales notes, if available.

## Instructions

1. List all roles that can advance, slow, block, or approve the decision.
2. Define what each role cares about.
3. Define the proof each role needs.
4. Identify the champion and likely blockers.
5. Choose one enablement asset that would help the champion sell internally.
